# OmniAide

> Your Omni-present Aide.

一个完全私有化部署的个人 AI 助手平台。融合全栈存储、事务管理与厚 AI 智能体，能感知你的情境，预判你的意图，并主动为你打点数字生活。你的文件、音乐、文档、视频、待办，以及每一天的行为足迹，都在这座由你完全掌控的私有服务器上，由一个懂你的 AI 管家温柔打理。

## 愿景

OmniAide 不是一个冷冰冰的工具箱，而是你数字世界里的专属管家。
它通过全维度的事件捕获理解你的习惯，借由双层记忆系统沉淀你的知识，并依靠主动智能引擎在恰好的时机递上恰好的服务。
所有数据只属于你，运行在你自己的机器上。隐私、智能、全能，一个都不妥协。

## 核心能力

📁 统一文件管理
文档、图片、音乐、视频……全都存储在自建的 MinIO 里，通过统一的标签、搜索、预览界面轻松驾驭。

🎵 智能歌单 & 媒体库
根据心情、天气或工作节奏自动生成播放列表，支持在线流媒体播放，让音乐随你的状态流动。

📝 待办与事务闭环
用自然语言一句话创建待办：“周五前完成项目评审并关联这个文件”。AI 会自动解析时间、优先级并附上附件。

🧠 长期记忆与知识图谱
对话、文件、网页、操作历史……OmniAide 不仅记得，还会抽象出你的兴趣、习惯和关系，形成你自己的语义记忆网络。

👁️ 全维度事件捕获（Aide-Eye）
桌面客户端 + 浏览器扩展，安静地记录应用切换、键入节奏、文件操作等元行为。所有数据端到端加密，本地脱敏后仅上传不可逆特征，你的隐私永远是最高优先级。

⚡ 主动智能与预测
不等待指令。当你专注工作超过 45 分钟，它会温柔提醒；当你为汇报苦苦搜寻数据时，它已准备好相关的图表和文档。基于你的行为模式，预判你的下一个需要。

🤖 多步规划 & 自动化执行
复杂任务如“整理上个月的项目资料并归档”，AI 自行分解步骤、检索文件、移动存储并生成报告。高风险操作必须由你确认，低风险任务自动流转。

🔐 完全自托管 & 隐私掌控
所有组件通过 Docker Compose 在你的机器上运行。你可以随时查看、导出或删除任何被采集的数据，真正的数据主权。

## 技术栈

层级	技术
前端	React + Ant Design / Zustand
后端	Python FastAPI + SQLAlchemy
AI 编排	LangChain / LangGraph + Ollama（本地模型）
对象存储	MinIO
关系数据库	MySQL 8.0
向量数据库	Qdrant
事件流与任务	Redis Streams + Celery
图数据库（可选）	Neo4j
客户端	Electron + Chrome Extension
部署	Docker Compose

## 快速开始

> 前置要求：Docker & Docker Compose，以及一块至少 8 核、32GB 内存的机器（若使用本地 LLM 推荐 16GB+ 显存 GPU）。

```bash
# 克隆仓库
git clone https://github.com/yourname/OmniAide.git
cd OmniAide

# 配置环境变量（可参照 .env.example）
cp .env.example .env

# 启动全部服务
docker compose up -d

# 初始化数据库 & 创建管理员用户
docker compose exec backend python manage.py init
```

启动后，访问 http://localhost:3000 进入 Web 界面。
建议同时安装 Aide-Eye 桌面客户端与浏览器扩展，以获得完整体验。

详细的部署、配置以及开发指南，请查阅 文档。

### 名字背后的含义

OmniAide = Omni（全能、全栈、全维度） + Aide（亲密助手）。
它既代表了你为个人打造的、涵盖一切数字生活的全栈技术基座，也代表了一个时刻相伴、熟知你习惯、主动服务而不是被动等待命令的数字伙伴。
